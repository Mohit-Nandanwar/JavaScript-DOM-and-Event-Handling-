# JavaScript-DOM-and-Event-Handling-
DOM (Document Object Model) is a programming interface for web documents.


# Interactive Web Page (HTML, CSS, JavaScript)

## 📌 Project Overview
This project demonstrates **basic DOM manipulation and event handling** using HTML, CSS, and JavaScript.  
The web page includes:
- A **header** with a greeting message.
- Four **interactive boxes** labeled Red, Blue, Green, and Yellow.
- An **input field** for the user’s name.
- A **button** to update the greeting dynamically.

---

## 🎯 Features
- **Greeting Update:**  
  When the user enters their name and clicks the button, the header changes from `Hello` to `Hello, [name]`.

- **Box Color Change:**  
  Clicking on a box changes its background color to the respective color mentioned in the box.
